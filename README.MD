# INSTRUCTIONS

## 1. Clone the repository

## 2. go to backend folder and run the following commands
```bash
pip install django djangorestframework
```

## 3. make migrations
```bash
python manage.py makemigrations
```

## 4. migrate
```bash
python manage.py migrate
```

## 5. run the server
```bash
python manage.py runserver
```
